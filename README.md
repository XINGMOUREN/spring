# 目前进度:日本!
## 😒 这是什么？
一个我的世界电子画册生成器,原理类似于AI绘画,程序会提供给你世界各地的旅游景点,或者我自己写好的,你只需要导入皮肤,自定义成就,就可以自动生成多组照片~~但是对电脑算力有要求~~(这是一给给我好哥们李春迎专门定制的项目,我可能不会对源代码进行大量简易化处理,有能力的可以自己研究)
## ✨ 有什么用？
生成电子画册,用来送人,具体功能有空详细写写。

- ✅ 使用 Vue.js 全家桶开发
- 🔴 可生成100+景点照片
- 📺 支持自定义皮肤
- 📃 支持自定义成就
- 📻 支持调节生成图片质量
- 🚫 支持图片生成尺寸调节
- 🛠 更多特性开发中
